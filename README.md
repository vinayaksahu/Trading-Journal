# FX Journal 1.0

A premium, local-first trading journal designed for XAUUSD, SMC (Smart Money Concepts), and ICT traders. 

## 🚀 Key Features

* **SMC/ICT Leak Finder & Confluences:** Easily track confluences (HTF Bias, Orderblocks, Fair Value Gaps, Liquidity Sweeps, BOS/MSS, Killzones) and find where you are leaking capital.
* **Premium Aero Frosted-Glass Aesthetics:** Elegant dark, light (Slate Paper), midnight, forest, and sapphire color palettes.
* **Advanced Portfolio Analytics:** Isolated account profiles, equity curve progression charts, Sharpe Ratio, Expectancy, and Kelly Criterion.
* **Unified Scorecard & Day Modal:** Interactive calendar view and scorecard to easily prep, review, and navigate daily notes.
* **Offline-First Security:** All trading logs, reviews, adjustments, and settings are stored locally in your browser (`localStorage`).
* **Web Audio Sound Effects:** Dynamic arpeggiated audio feedback for wins, losses, and drawdown lockouts.

## 📂 Repository Structure

* `index.html`: The main, self-contained application file. Open this in any browser to use the journal.
* `README.md`: Project documentation.
* `/archive/`: Directory containing legacy version files (V8, V9, V10, and backup configurations).

## 💻 How to Run Locally

Simply clone the repository and open `index.html` in your browser:
```bash
git clone https://github.com/vinayaksahu/Trading-Journal.git
cd Trading-Journal
start index.html # Or open with any web browser of choice
```
